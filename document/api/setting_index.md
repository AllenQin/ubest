设置页面接口
===============================

[<<返回API文档列表](../API.md)

##### 接口地址 http://ubest.allenqin.com/index.php?r=setting/index

##### 请求方式 GET

##### 请求参数

|参数        | 类型   |  是否必填 | 示例 |  参数含义 |
|--------   | ------:| --------:| ------:| -----: |

##### 返回结果
```
{
    "code": 200,
    "data": {
        "sexual": 0, // 性取向 0:女 1:男
        "msg_tip": 1 // 聊天通知 0:关闭 1:开启
    },
    "msg": ""
}
```
##### 返回状态码
|状态码|含义|
|--------|-------------------:|
|200 | 数据正常 |
|1003|请求方式不正确|
|1002| 参数错误, 具体错误信息 msg字段返回|
|1001| 需要用户登录|
